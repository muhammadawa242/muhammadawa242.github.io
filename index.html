HTML, CSS and Javascript

Hypertext Markup Language(HTML)

It is not a programming language rather it is used to develop the basic infrastructure of a website.Two main features of HTML are tags and the attributes.

Cascading Style Sheets(CSS)

Almost all the tags in HTML support an attribute called style. These attributes define the style of different components of various tags including Font-size or the background-color of the page.

But this will be an exhausting work for you to define the style of each single tag even if you want to use the same style-format for different tags. By using CSS you can reduce the distractions caused by these style attributes.

Javascript(javascript)

Javascript is a proper programming language. Its syntax is quite simillar to C while most of its features are simillar to Python. It is an object-oriented programming language which means that it still contains loops, conditions and variables etc like C and others but mostly you have to deal with the objects and the whole story revolves around these objects which contain methods and properties associated to them.
